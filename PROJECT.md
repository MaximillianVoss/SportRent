# SportRent

## Тема

Разработка мобильного приложения для аренды спортивного инвентаря.

## Описание

Курсовой проект на .NET MAUI с локальной SQLite-базой данных. Приложение демонстрирует пользовательские сценарии аренды спортивного инвентаря: вход в аккаунт, просмотр каталога, карточка товара, оформление аренды, просмотр заказов и личный кабинет.

## Состав

- `SportRent.sln` - решение Visual Studio.
- `SportRent.Mobile` - мобильное MAUI-приложение с экранами, сервисами и ViewModel.
- `SportRent.Data` - EF Core сущности и `SportRentDbContext`.
- `SportRent.DbTool` - консольный инструмент для сборки SQLite-базы из SQL-скриптов.
- `database` - SQL-скрипты создания схемы и заполнения демо-данными.
- `docs` и `output/doc` - материалы пояснительной записки и итоговые документы.

## Технологии

- C# / .NET MAUI
- SQLite
- Entity Framework Core
- MVVM
- Python-скрипты для документации

## Статус

Актуальный учебный проект с кодом, базой данных и документами. Текущее состояние зафиксировано в git; подробное описание также есть в `README.md`.
